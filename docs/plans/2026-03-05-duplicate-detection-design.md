# 重複検知機能 設計書

## 概要

異なるfolderハッシュ間で「同じ楽曲パッケージ」と思われるものをファジーマッチングで検出し、専用ビューで一覧表示する機能。

## スコープ

- **今回**: スキャン実行 → 重複グループの閲覧
- **スコープ外**: ファイル移動・マージ・削除操作（将来拡張）

## アルゴリズム

### ブロッキング（候補グループ生成）— SQL側

artist正規化（小文字化+トリム）でグループ化し、2つ以上のfolderが存在するartistグループのみを候補として抽出する。
加えて、artist不一致でもtitle完全一致のfolder同士も候補に含める。

### 類似度計算 — Go側（グループ内のみ）

同一norm_artistグループ内のfolder同士を比較する。

| フィールド | 類似度計算 | 重み |
|-----------|-----------|------|
| title | 前方一致文字数 / max(len(a), len(b)) | 0.4 |
| artist | 前方一致文字数 / max(len(a), len(b)) | 0.3 |
| genre | 前方一致文字数 / max(len(a), len(b)) | 0.1 |
| BPM | 範囲が重なれば1.0、そうでなければ0.0 | 0.2 |

- 総合スコア = 加重平均（0.0〜1.0）→ %表記（0〜100）
- 閾値: 60%以上を重複候補として返す（固定値）

### パフォーマンス

ブロッキングによりartist正規化グループ内のみ比較するため、10,000曲規模でも問題ない。

## API設計

### エンドポイント

```go
func (a *App) ScanDuplicates() ([]DuplicateGroup, error)
```

### レスポンス型

```go
type DuplicateGroup struct {
    ID      int               // グループ番号
    Members []DuplicateMember
    Score   int               // グループ内の最高類似度（%）
}

type DuplicateMember struct {
    FolderHash string
    Title      string
    Artist     string
    Genre      string
    MinBPM     float64
    MaxBPM     float64
    ChartCount int
    Path       string // 代表パス（フォルダまで）
}
```

## フロントエンドUI

### タブ追加

既存の3タブ（楽曲一覧・譜面一覧・難易度表）に「重複検知」タブを追加。

### レイアウト（上下SplitPane）

```
┌─────────────────────────────────────────────┐
│ [楽曲一覧] [譜面一覧] [難易度表] [重複検知]  │
├─────────────────────────────────────────────┤
│ [スキャン実行] ボタン    結果: 12グループ     │
├─────────────────────────────────────────────┤
│ 重複グループ一覧（上ペイン）                  │
│                                             │
│  92% タイトルA  |  85% タイトルB  |  78% ... │
├─────────────────────────────────────────────┤
│ グループ詳細（下ペイン）                      │
│                                             │
│ フォルダ1: G:\BMS\event1\...                │
│  title / artist / genre / BPM / 譜面数       │
│                                             │
│ フォルダ2: G:\BMS\event2\...                │
│  title / artist / genre / BPM / 譜面数       │
│                                             │
│ 類似度内訳: title 95% / artist 90% / ...    │
└─────────────────────────────────────────────┘
```

- 上ペイン: グループ一覧（スコア降順ソート）
- 下ペイン: 選択グループのメンバーをカード形式で並べ、各フィールドの類似度内訳を表示
- スキャン未実行時は「スキャン実行」ボタンのみ表示

## 制約

- 結果の永続化はしない（songdata.dbが外部で変わりうるため、毎回スキャンで再計算）
- 閾値はデフォルト60%固定（UIにスライダーは設けない）
