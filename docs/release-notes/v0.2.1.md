## 新機能
- LR2IRに登録されている全MD5（約33万件）のIR情報を事前取得するCLIコマンド（`prefetch-ir`）と、取得済みelsa.dbをユーザーのelsa.dbへ統合するCLIコマンド（`merge-db`）を追加。中断・再開やリクエスト間隔制御にも対応。
- ビルド成果物にIR事前取得済みのelsa.dbを同梱し、初回起動直後からIR情報を参照できるようにした。
- 差分導入画面のファイル名・推定先セルから対象フォルダを直接開ける「フォルダを開く」アイコンボタンを追加し、各画面で利用していたボタンも共通コンポーネント（OpenFolderButton）に統一。

## バグ修正
- 差分導入のファイル移動でドライブをまたぐ移動（例: `C:` → `G:`）が失敗する問題を修正。`os.Rename`方式からコピー＋削除方式に変更し、クロスドライブ移動に対応。
- Shift-JISエンコードのBMSファイルでTITLE/ARTISTなどが文字化け（◆◆◆）して表示される問題を修正。BMSパーサーにShift-JIS→UTF-8変換を追加。
- 差分導入の推定スコアが5点以下のときに、低信頼の推定先が誤って候補として表示されてしまう問題を修正。スコア5以下は推定失敗扱いとして除外。
