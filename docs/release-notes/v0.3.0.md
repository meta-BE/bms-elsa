## 新機能
- 重複検知のスコアリングにMinHash（WAV定義集合の類似度）を50%の重みで統合し、文字列メタデータだけでは判定しづらかった重複を高精度に検出できるように改善。重複詳細画面ではWAV定義類似度の表示と「フォルダを開く」ボタンも追加。
- ビルド版で動作するカスタムコンテキストメニュー（カット/コピー/ペースト/削除）を追加。Wails v2 + Vite 5環境で右クリックメニューが出ない問題を解消。
- 一覧テーブルの表示を改善。Title/Artist/Genreカラムが残余幅に応じて伸縮するflex幅分配、IR/URLの中央揃え・Level/Notes/Year/Chartsの右揃え、ヘッダーのスクロールバー幅補正を導入。

## バグ修正
- IR一括取得ボタン（BulkFetchButton）が他ビューのIR取得イベントにも反応してしまう不具合を修正。自身が開始していないインスタンスは進捗・完了イベントを無視するようガードを追加。
