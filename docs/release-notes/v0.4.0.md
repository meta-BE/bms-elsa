## 新機能
- 重複検知画面からBMSフォルダ同士をマージ（統合）できる機能を追加。サブディレクトリを含めて再帰的にファイルを移動し、競合時はファイル作成日時を比較して新しい方を残す。マージ確認はDaisyUIダイアログで行い、エラーはAlertModalで画面中央に表示（error/warningレベルをバッジで色分け）。マージ後はグループ・メンバーを画面上から自動的に除去する。
- 設定画面に「ファイル別ログ出力」チェックボックスを追加。フォルダマージや差分導入時に、個別ファイルの移動ログを記録するかどうかを切り替え可能にした。
