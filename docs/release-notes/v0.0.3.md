## 新機能
- 譜面の導入先推定機能を追加。難易度表エントリに対して、ローカルの譜面フォルダから導入先候補を自動推定し EntryDetail に推定カードとして表示する。タイトル完全一致・本体 URL 一致に加え、末尾の `[...] / (...) / -...-` を除去したベースタイトル一致とアーティスト一致を組み合わせ、4 手法のスコア合算で候補をランク付けする。
- フォルダを開く機能を追加。SongDetail / ChartDetail / EntryDetail（導入済みのみ）のヘッダーから OS のファイルマネージャでフォルダを開ける。
- 譜面情報・IR 情報を共通コンポーネント（ChartInfoCard / IRInfoCard）に抽出し、3 つの詳細パネルで共有。SongDetail の譜面選択時に譜面情報＋ IR 情報を両方表示するよう改善し、譜面情報にインストール先ファイルパスを表示。

## バグ修正
- `OpenFolder` のエラーハンドリングを `os.IsNotExist` のみから `err != nil` 全般に変更し、パーミッション不足やシンボリックリンク破損も捕捉するよう改善。
- レートリミットテストの閾値を 0.5 秒に調整して flaky を解消。
