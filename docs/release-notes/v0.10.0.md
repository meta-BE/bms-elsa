## 新機能
- URL 書き換え（リライトルール）の適用方式をフロントエンド表示時適用に変更。IR 情報カードの本体URL・差分URL・備考欄のリンクが、表示のたびにリライトルールを適用したうえで表示されるようになり、ルールを編集すれば即座に表示へ反映される。これにより、IR 備考欄に複数 URL が含まれていても、すべてのリンクに対してルールが適用されるようになった。
- 起動時およびルール編集時にリライトルールのストアを更新し、各ビューが常に最新ルールで表示できるようにした。

## バグ修正
- linkify の URL 検出正規表現を元の形式に戻し、Web Archive のように URL 内に `http://` を含むケースで誤分割されていた問題を修正。

## その他の変更
- 旧方式の「動作URL推定」機能（バックグラウンドで `working_body_url` / `working_diff_url` を DB に保存する処理）と、設定画面の動作URL推定セクションを廃止。`chart_meta` テーブルから `working_body_url` / `working_diff_url` カラムを削除するマイグレーションを追加。
