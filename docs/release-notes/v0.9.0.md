## 新機能

- 重複検知の MD5 完全一致対応
  - 同一 MD5 が複数フォルダに存在するケースを確定的な重複として検出するようになりました。スコアは 100% で表示され、重複詳細では MD5 一致であることが分かるよう表示されます。
  - スキャンは「MD5 完全一致 → ファジーマッチ」の2段階パイプラインで実行され、MD5 一致で確定したペアにはファジースコア計算を行いません。
- 重複検知スキャンのバックグラウンドタスク化
  - 重複検知スキャンが、他のタスクと同様に起動時（MinHashスキャン完了後）に自動実行されるようになりました。
  - 重複検知ビューはバックグラウンドスキャン結果を自動で表示するようになり、ビュー内のスキャンボタンは廃止されました。
  - 基本設定モーダルに4つ目のバックグラウンドタスクとして重複検知スキャンの状態が表示されるようになりました。
