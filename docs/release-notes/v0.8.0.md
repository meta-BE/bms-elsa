## 新機能

- 起動時バックグラウンドタスク自動実行
  - MinHashスキャンと難易度表一括更新をアプリ起動時に自動実行するようになりました。譜面一覧の MinHash スキャン起動ボタンは廃止されました。
  - 動作URL推定（書き換えルール適用）も起動時に自動で実行されるようになりました。譜面一覧・難易度表ビュー上の動作URL推定ボタンは削除されました。
- 基本設定モーダルでバックグラウンドタスクの状態確認
  - 基本設定モーダルに各バックグラウンドタスク（MinHashスキャン・難易度表更新・動作URL推定）の進捗表示セクションが追加されました。
  - 各タスクの完了サマリー（計算/スキップ/失敗件数、成功件数や失敗テーブル名、適用/スキップ件数など）が表示されるようになりました。

## バグ修正

- 動作URL推定の完了サマリー表示を3タスク間でUI表示が統一されるように調整しました。
