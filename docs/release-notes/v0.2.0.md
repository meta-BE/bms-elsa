## 新機能
- 差分導入タブを追加。BMS/BME/BMLファイルをドラッグ&ドロップすると、MinHash類似度とメタデータ（タイトル/本体URL/アーティスト）を統合したスコアで導入先フォルダを自動推定し、ファイルを移動できる。
- BMSパーサーを拡張し、ヘッダー（TITLE/SUBTITLE/ARTIST/SUBARTIST/GENRE）とファイル全体のMD5ハッシュもパースする`ParseBMSFile`を追加。
- MinHash類似度検索（Go全件スキャン方式）を実装し、推定先候補のランキングに利用。
- 汎用ファイル移動ユーティリティを追加。

## バグ修正
- 差分導入の推定先パス表示を修正。macOS上でもWindows形式パス（`\`区切り）を正しく扱えるようにし、推定方法のラベル（WAV定義/本体URL/タイトル）と統合スコア（最大10点）の表示を整理。
- ドラッグ&ドロップによるBMSファイル受け取りが動作しない不具合を修正し、`OnFileDrop`を`App.svelte`に集約してアクティブタブごとに振り分けるよう改善。
