## 新機能
- 楽曲詳細パネルに「フォルダ移動」ボタンを追加。OSネイティブのディレクトリ選択ダイアログで指定した移動先に楽曲フォルダを移動できる。確認ダイアログで誤操作を防ぎ、移動完了後は移動先パスとファイル数を結果ダイアログで表示する。同一ファイルシステムではrenameでアトミックに移動し、クロスファイルシステム時はコピー＋削除にフォールバックする。移動済みの楽曲はセッション中のみ楽曲一覧で黄色背景＋「移動済み」バッジで表示される。
- コンテキストメニューにリンク用アクション（「開く」「URLをコピー」）を追加。リンク上で右クリックするとシステムブラウザで開いたり、URLをクリップボードにコピーできる。
- LR2IR備考欄の表示を改善し、`<br>`を改行として反映、URLを自動的にハイパーリンク化（XSSエスケープ付き）するようにした。

## バグ修正
- macOSでLANG環境変数が未設定の場合に、クリップボード（pbcopy）経由のコピー結果が文字化けする問題を修正した。
